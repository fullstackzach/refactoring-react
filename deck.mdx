# Higher Order Components

What are they?

🤝 Another way to Share common functionality among components

👎 Mixins used to be the thing (thats so 2015 now)

🔥 React ditched React.CreateClass in favor of native ES6 classes

---

You might recognize these

```
export default withRouter(MyComponent)
export default withTheme(MyComponent)
```

Or this iconic one

```
export default connect(
  mapStateToProps, mapDispatchToProps
)(MyComponent)
```
